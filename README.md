# 📊 Netflix Movies Analysis using Python

A complete data analytics project demonstrating data cleaning, exploratory data analysis (EDA), visualization, and business insights using the Netflix Movies and TV Shows dataset.

## Technologies Used

- Python
- Pandas
- Matplotlib
- VS Code

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Visualization
5. Business Insights
6. Report

## Key Insights

- Movies dominate Netflix's catalog.
- The United States contributes the most content.
- Drama is the most common genre.
- Netflix content grew rapidly after 2015.
- TV-MA is the most common content rating.
